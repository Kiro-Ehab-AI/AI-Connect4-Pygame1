# AI Connect-4 Game 🎮

A fully interactive Connect-4 desktop application built with Python and Pygame. This project features a decision-making AI opponent powered by the Minimax algorithm.

## Features ✨
* **Graphical User Interface:** Built a responsive and interactive board using Pygame.
* **Smart AI Opponent:** Implemented the Minimax algorithm with a custom heuristic evaluation function.
* **Adjustable Difficulty:** Users can select the AI's search depth (Levels 1 to 6) before starting the game.
* **Real-time Analytics:** A side panel displays real-time statistics including the AI's chosen column, heuristic score, evaluated nodes, and decision time (ms).
* **Game Summary:** Displays total moves and final depth upon game completion.

## Technologies Used 🛠️
* Python
* Pygame
* NumPy
* Math & Time modules

## How to Play 🚀
1. Run the `Connect_4_GUI.py` script.
2. Select the AI difficulty depth on the start screen.
3. Use the mouse to drop your pieces (White) and try to beat the AI (Red)!****
